# DavyNews

# 项目介绍
DavyNews是一款基于 MVP + Retrofit2 + Rxjava + dagger2 开发的新闻类资讯类项目，可以在上面阅读新闻、图片、视频。
本项目使用 Api 来自凤凰新闻，煎蛋，此开源项目提供学习交流使用,如果你觉得不错, 对你有帮助, 欢迎点个 star 谢谢
